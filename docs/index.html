What it does

Voice dictation, cleaned up by an on-device LLM.

OpenType is a macOS menu bar app for AI-powered voice input. Speak naturally and polished text is typed at your cursor in any app — with speech recognition and language model cleanup running locally on Apple Silicon.

  • On-device speech recognition

    WhisperKit, Apple Speech, Qwen3-ASR, or MiMo run locally on Apple Silicon. No audio uploads required.

  • LLM text cleanup

    Local MLX Qwen2.5/Qwen3 or a remote LLM removes fillers, fixes grammar, and formats lists based on context.

  • Screen context OCR

    ScreenCaptureKit + Vision reads on-screen text so the LLM can pick the right homophone from what you are looking at.

  • Voice command mode

    Speak a command; OpenType uses your screen as context to summarize, reply, translate, or rewrite.

  • Works in any app

    A global hotkey plus Accessibility-based text insertion puts the result at your cursor everywhere.

  • Bilingual & private

    English and Simplified Chinese UI. Audio and transcription stay on-device by default; remote providers are optional.

Private by default

FAQ

Questions, answered.

What is OpenType?

OpenType is a native macOS menu bar app that turns your voice into polished text in any app. Hold a hotkey, speak, and the transcribed, cleaned-up text is typed directly at your cursor. Speech recognition and language model cleanup run locally on Apple Silicon by default, with optional remote LLM providers.

Does OpenType work offline?

Yes. With the WhisperKit or Apple Speech engine plus a local MLX model such as Qwen3, OpenType transcribes and refines text entirely on-device — no audio leaves your Mac. Remote LLM providers are optional and only used if you configure them.

Which macOS version and hardware does OpenType require?

macOS 26 (Tahoe) or later on Apple Silicon (M1, M2, M3, or M4). Intel Macs are not supported.

How is OpenType different from Apple's built-in dictation?

OpenType adds on-device LLM cleanup (filler removal, grammar fixes, structured formatting), screen context via OCR to correct homophones, a voice command mode that acts on what is on screen, and works in any app from a global hotkey without switching context.

Which apps does OpenType work in?

Any macOS app that accepts text input. OpenType uses the Accessibility API to insert text at your cursor, with a clipboard-paste fallback.

Is OpenType free and open source?

Yes. OpenType is MIT-licensed and the full source is on GitHub. Prebuilt binaries are published on GitHub Releases.

Which languages does OpenType support?

Speech recognition supports many languages via WhisperKit and Apple Speech. The app UI is bilingual (English and Simplified Chinese). The LLM cleanup language is configurable independently of the recognition language.

How does screen context help dictation?

OpenType can capture on-screen text with ScreenCaptureKit and the Vision framework, then feed it to the LLM so it picks the correct homophone (for example, "there" vs "their") based on what you are looking at.
